Необходимо реализовать веб-приложение для хранения информации о
фильмах. Вместе с реализацией требуется предоставить документ
объясняющий архитектуру приложения. Желательно использование ООП
подхода.

Информация о фильме:
1. Уникальный идентификатор
2. Название
3. Год выпуска
4. Формат (VHS, DVD, Blu-Ray)
5. Список актеров (“Имя и фамилия актера”)

Функции, которые должна поддерживать система:
1. Добавить фильм
2. Удалить фильм
3. Показать информацию о фильме
4. Показать список фильмов отсортированных по названию в алфавитном порядке
5. Найти фильм по названию.
6. Найти фильм по имени актера.
7. Импорт фильмов с текстового файла (файл прилагается). Файл должен
загружаться через веб-интерфес.

Задание необходимо реализовать на PHP или Perl без использования
фреймворков. Шаблонизаторы использовать разрешено. Данные должны
храниться в базе данных MySQL.
